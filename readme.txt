== Description ==

**Advanced Facebook Page Promoter Lightbox** is an facebook page promoter plugin. It offers a quicker & flexible way to show a facebook page promoter popup / lightbox in your website. You can customize the lightbox styles, can disable for logged in user, . 

There are numerous plugins for facebook page promotion lightbox but This plugin will help you to show lightbox in any type of devices like, mobile, desktop, tab, ipad, etc. If you want to hide this for loggedin user you can do this. If you want to show it once in a day or want to hide facebook friends faces or want to hide cover photo of facebook page it's better choice. You can customize this plugins background and lightbox colour according to your requirement or site design. You can also set a width, height of lightbox. It will helps you to get more followers in your facebook page.

Here are some **features** that come with this plugin:

1. The plugin helps to add lightbox with facebook page likebox.
2. You can hide popup to logged in user.
3. You can set frequency to show popup in a day.
4. You can customize lightbox styles, background, color according to your requirement.
5. Set width and hight of lightbox.
6. Easy to user advanced admin panel, with greate ui design.
7. Lightbox effect is ZoomOut Effect.
8. Close popup on overlay click.
9. Increase your facebook page by showing this lightbox with your facebook page likebox.
10. Hide facebook friends faces in likebox.
11. Hide cover photo of your facebook page in likebox.
12. Customize Likebox Title
13. Hide / Show popup in desired pages
14. Set appearance time of lightbox

== Installation ==

1. Unzip the downloaded plugin and upload the folder "advanced-facebook-page-promoter-lightbox" to the "/wp-content/plugins/" directory.
2. Go to the plugin menu in WordPress and activate the plugin.
3. Go to the settings menu >> FB Lightbox Settings and configure the options.
4. See Screenshots for more information
